# Demo task 1

How to run from scratch:

1. Get docker
2. 
        make run
    
The app will be available through: 
`http://dashboard.localhost`

If you're on a Mac, add this host
to your `/etc/hosts`

# About

App based on the codingmachine template 
for Symfony 4 + Vue.JS . Check out @ github

Stack:
* MySQL
* PHP 7.4
* SF4
* Vue.JS
